# 11ty Vue Starter

A modern, production-ready starter for building static sites with 11ty, Vue 3, Vite, and comprehensive testing.

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production  
npm run build
```
